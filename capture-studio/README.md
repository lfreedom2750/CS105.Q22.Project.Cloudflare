# Capture Studio

Thu muc nay la viewer rieng de chup anh model/doi tuong dua vao bao cao mon Do hoa may tinh.

## Cach chay

Tu thu muc goc project:

```bash
npm run dev
```

Sau do mo:

```text
http://localhost:5173/capture-studio/
```

## Cach dung

- Chon doi tuong trong danh sach.
- Chon `Canh chup`:
  - `Studio rieng` de chup model tach nen.
  - `Canh game - Mua ...` de dat model len dung moi truong chay cua game.
- Keo chuot de xoay, cuon chuot de zoom.
- Chon nen studio, nen toi, nen am vang hoac PNG trong suot.
- Chon do phan giai xuat anh.
- Bam `Tai PNG`.

Anh se duoc tai ve voi ten theo doi tuong va do phan giai.

## Goi y anh cho bao cao

- Dung `Studio sang` cho model co mau toi.
- Dung `Nen toi` cho vat the sang, portal, coin, dragon.
- Dung `PNG trong suot` khi muon chen model vao Word/PowerPoint.
- Dung `Canh game - Mua ...` khi muon chup model nam tren lane duong, texture duong, cay, fog va anh sang giong game.
- Dung 2560 x 1440 cho anh 16:9 trong bao cao.
- Dung 2000 x 2000 cho anh model don.
